Little Sweet Bakery

×

Little Sweet Bakery

An informative and online shopping webpage. Google map API is used to show location. AJAX call is used to get the data from JSON to the webpage. All the products’ data, products’ information, is dynamically loaded using JavaScript and ES6 Syntax. The webpage has functionalities to display a star rating for each product, to show product information individually, to add a product to shopping basket, to adjust items in shopping basket and to search products from search box. Simple PHP Post is used for the contact form in the webpage. SCSS clip-path and shape-outside properties are used to display pictures that are, ‘out of the box format’. The web page is device responsive. User-friendly front-end features are used.

  • HTML
  • SCSS
  • JAVASCRIPT
  • JQUERY
  • PHP

French Quiz

×

French Quiz

The idea of the project is to help adults learning a new foreign language, in this case French. The test includes 5 questions which are randomly generated from the question banks. Each question has 4 mutiple choice answers. Users submit their choice for each question and then discover if their choice is correct. A correct answer is indicated in green with a tick symbol and an incorrect answer is indicated in red with a cross symbol. After completeting 5 questions, a user will get the end result.
